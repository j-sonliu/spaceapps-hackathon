High-Energy Astrophysics Telemetry Viewer

About

This purpose of this platform is to inform people of the locations of high-energy celestial phenomena. In this platform, we can see these celestial phenomena in the following wavelengths: Gamma ray, X-ray, Ultraviolet, AllWISE, Infrared, and radio. To locate these high-energy celestial phenomena, we use a celestial coordinate system. The system that is used to find the location of these celestial phenomena is the J2000. This is a system that is used to define a celestial reference frame. Now how do celestial coordinates work?


Imagine a big sphere encapsulating the Earth. That would be the celestial sphere. Celestial coordinates work by using the celestial sphere. What's on the celestial sphere are the high-energy celestial phenomena that occurs out there in space. The celestial sphere has a an equator called the celestial equator which is not the same as Earth's equator. It has a north pole called the north celestial pole which is the same as Earth's north pole. And a south pole called the south celestial pole which is the same as Earth's south pole.
